# Video Recommendation Site Demo

Demo repository for a Video Recommendation Site. This is a simple demonstration of video recommendation functionality using HTML, CSS, and JavaScript.

## Features

- Browse video catalog
- View video recommendations based on selected videos
- Responsive grid layout
- Simple recommendation algorithm based on categories and tags

## Getting Started

Simply open `index.html` in a web browser to see the demo in action.

```bash
# Clone the repository
git clone https://github.com/kiminus/video-rec-public.git
cd video-rec-public

# Open in browser
open index.html  # macOS
# or
xdg-open index.html  # Linux
# or simply double-click index.html in your file explorer
```

## Demo Features

- Video grid with thumbnails
- Click on any video to see recommendations
- Categories: Technology, Science, Entertainment, Education
- Simple recommendation engine based on category matching and content tags

## Architecture

- `index.html` - Main demo page
- `styles.css` - Styling for the video recommendation interface
- `app.js` - Video data and recommendation logic
